### Gateway Security

- **Token auth:** Set `OPENCLAW_GATEWAY_TOKEN` for all connections
- **Binding:** Defaults to loopback (127.0.0.1) â€” use LAN only when necessary
- **Pairing:** Device-based approval required for non-local connections

}
