# aaa-penang-weaver

## EUREKA Design

**Penang** = Your roots, your grounded knowledge base.

This weaves your PKB (Obsidian/git) as a **thermodynamic system** — new information is anchored, connected, and entropy is reduced through linking. Orphan notes = high entropy = must link or delete.

## Human Context

You maintain:
- Obsidian vault (git-backed)
- Daily notes (YYYY-MM-DD.md)
- Long-term MEMORY.md
- Constitutional frameworks, energy research, personal reflections

You don't need "note-taking." You need **entropy reduction** in your second brain.

## Atomic Composition

| Stage | Action | Function |
|-------|--------|----------|
| 000_INTAKE | **ANCHOR** | New information (article, book, conversation) |
| 333_DELIBERATE | **INTEGRATE** | Connect to existing nodes (Thermodynamics? Governance?) |
| 222_PROCESS | **REASON** | Find contradictions: "You said X, now Y — reconcile?" |
| 888_HOLD | **FORGE** | Refine into atomic zettels (one idea, one file) |
| 777_VERIFY | **AUDIT** | Check link entropy (orphan nodes = high entropy) |
| 999_EMIT | **SEAL** | Auto-commit with entropy score in message |

## Usage

```

/action penang-weaver integrate source="article.md"
/action penang-weaver audit=entropy

```

## Entropy Metrics

| Metric | Target | Action if Exceeded |
|--------|--------|-------------------|
| Orphan nodes | < 5% of total | Link or delete |
| Dead links | 0 | Fix immediately |
| Avg links per note | > 2 | Improve connectivity |
| Daily entropy Δ | < 0 | Must reduce daily |

## Daily Workflow

1. **Morning**: ANCHOR new inputs (readings, thoughts)
2. **Afternoon**: INTEGRATE + REASON (connect, find contradictions)
3. **Evening**: FORGE (refine zettels) + AUDIT (check entropy)
4. **Night**: SEAL (git commit with entropy score)

## Governance Audit

- **Ω₀**: Computed from orphan rate + contradiction count
- **F4 Clarity**: One idea per zettel
- **F7 Humility**: Contradictions surfaced, not hidden
- **F10 Ontology**: Categories respected
- **Status**: DOMAIN-SPECIFIC FORGE from atomic primitives
